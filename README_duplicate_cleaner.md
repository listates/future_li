# 重复文件清理工具

## 功能
这个工具可以帮助你找到并删除电脑上的重复文件，只保留一份。

## 使用方法

### 方法1：使用批处理文件（最简单）
1. 双击运行 `clean_duplicates.bat`
2. 选择扫描选项：
   - 1: 扫描当前目录
   - 2: 扫描指定目录
   - 3: 退出
3. 查看扫描结果
4. 按照提示删除重复文件

### 方法2：使用PowerShell脚本（更灵活）
```powershell
# 1. 先扫描查看重复文件（不删除）
.\find_duplicates.ps1 -Path "C:\Your\Directory"

# 2. 确认无误后删除重复文件
.\find_duplicates.ps1 -Path "C:\Your\Directory" -Delete
```

## 安全建议

1. **先备份重要文件**：删除前确保重要文件已备份
2. **先扫描查看**：使用 `-DryRun:$true`（默认）先查看哪些文件会被删除
3. **逐目录清理**：不要一次性扫描整个C盘，建议按目录分批处理
4. **检查结果**：仔细查看脚本输出的文件列表，确认无误后再删除

## 工作原理
脚本通过计算文件的MD5哈希值来识别重复文件，即使文件名不同，只要内容相同就会被识别为重复。

## 常见清理目录建议
- 下载文件夹：`C:\Users\<用户名>\Downloads`
- 桌面：`C:\Users\<用户名>\Desktop`
- 图片文件夹：`C:\Users\<用户名>\Pictures`
- 文档文件夹：`C:\Users\<用户名>\Documents`

## 注意事项
- 系统文件不要随意删除
- 程序文件（.exe, .dll等）谨慎处理
- 建议从用户数据目录开始清理
